HTML

  • The <html> element represents the root of an HTML page.
  • The <head> element contains meta-information about the HTML document.
  • The <body> element defines the document's body, including all visible contents.
  • The <a> element defines a hyperlink for linking between pages.
  • The <div> element is used for defining divisions or sections in a document.

CSS

  • CSS Margins: Learned about controlling the space outside of an element's border.
  • CSS Padding: Understood how to manage the space between an element's content and its border.
  • Inline CSS: Explored adding CSS directly within an HTML element using the 'style' attribute.
  • Internal CSS: Discovered how to embed CSS in the head of the HTML document using the <style> tag.
  • External CSS: Learned about linking separate CSS files to HTML using the <link> tag.
  • CSS Classes: Gained knowledge on how to define and apply CSS classes for styling HTML elements.

Git

  • git status displays the state of the working directory and the staging area, showing any changes that have been staged, untracked, or modified.
  • git checkout -b branch-name creates a new branch named 'branch-name' and switches to it immediately.
  • git clone [url] creates a clone or copy of a repository at the specified URL into a new directory.
  • git add -A adds all changes in the working directory to the staging area, including new files, modified files, and deleted files.
  • git commit -m "[commit message]" commits the staged changes to the repository with a descriptive message provided in quotes.
